## 二、测试环境准备

### 2.1 测试仓库准备

```bash
# 创建测试仓库
mkdir -p ~/test-worktree-manager
cd ~/test-worktree-manager
git init

# 创建初始提交
echo "# Test Repo" > README.md
git add .
git commit -m "Initial commit"

# 创建多个分支用于测试
git checkout -b feature/test-1
git checkout -b feature/test-2
git checkout -b hotfix/bug-123
git checkout main
```

### 2.2 测试数据

| 数据类型 | 准备内容 |
|----------|----------|
| 干净的 worktree | main 分支，无修改 |
| 有修改的 worktree | 创建文件修改，未提交 |
| 有冲突的 worktree | 模拟合并冲突 |
| Detached HEAD | checkout 到特定 commit |
