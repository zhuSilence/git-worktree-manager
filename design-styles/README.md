# Git Worktree Manager - Design Styles

本目录包含多种设计风格供选择，基于 [awesome-design-md](https://github.com/VoltAgent/awesome-design-md) 格式。

---

## 风格一览

| 风格 | 文件 | 特点 | 适合场景 |
|------|------|------|----------|
| **Default** | `DESIGN-default.md` | 现代深色主题，简洁高效 | 通用，推荐 |
| **Warp** | `DESIGN-warp.md` | 终端原生，等宽字体优先 | 开发者工具感 |
| **Linear** | `DESIGN-linear.md` | 极简紫色，大量留白 | 项目管理美学 |
| **Cursor** | `DESIGN-cursor.md` | AI 编辑器风格，渐变高亮 | AI 功能突出 |

---

## 如何使用

1. 选择一种风格
2. 复制对应的 `DESIGN-xxx.md` 到项目根目录并重命名为 `DESIGN.md`
3. 告诉 AI 代理："按照 DESIGN.md 的风格创建 UI"

---

## 风格对比

### Default（推荐）
- 深色背景 `#242424`
- Indigo 强调色
- 标准 Inter 字体
- 平衡的信息密度

### Warp
- 深黑背景 `#0d0d0d`
- 电蓝强调色
- 等宽字体广泛使用
- 终端/命令行美学

### Linear
- 深紫黑背景 `#0a0a0b`
- 紫色强调
- 大量留白
- 极简优雅

### Cursor
- 深黑背景 `#0f0f0f`
- 紫蓝渐变
- AI 功能特殊视觉
- 代码编辑器美学
